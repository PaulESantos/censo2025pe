# Población censada por autoidentificación étnica (Censo 2025)

Dataset ordenado con la población que se identificó con algún grupo
étnico.

## Usage

``` r
censo_etnicidad
```

## Format

Un objeto tibble con 2,268 filas y 7 columnas:

- departamento:

  Nombre del departamento.

- provincia:

  Nombre de la provincia.

- area:

  Área de residencia.

- sexo:

  Sexo de la población.

- grupo_edad:

  Grupos de edad.

- grupo_etnico:

  Etnia ("quechua", "aimara", "mestizo", "blanco", etc.).

- poblacion:

  Cantidad de personas.

## Source

Instituto Nacional de Estadística e Informática (INEI) - Censos
Nacionales 2025.
